🗺️ Project Milestones
5-Phase Build Workflow

Students progress through structured milestones — from setup to a fully featured platform.

01
Project Setup
Configure folder structure and create mock course data with realistic JSON datasets.
02
Course Pages
Build course listing and individual detail pages with title, description, and duration info.
03
Enrollment System
Implement enrollment logic and the "My Courses" dashboard for tracking enrolled content.
04
Progress Tracking
Add percentage completion, lesson checkboxes, and a visual progress bar simulation.
05
Polish & Persist
Search/filter, responsive design for mobile & desktop, and LocalStorage persistence.
